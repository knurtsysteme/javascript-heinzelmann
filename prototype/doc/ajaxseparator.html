Literal: AjaxSeparator

Download AjaxSeparator.js
since: 0.20061109
requires: prototype
author: Daniel Oltmanns
version: 0.20080823
copyright: KNURT Systeme
license: GPL 3
With this literal you can seperate a request in many pieces. This makes it possible, to seperate big sized server answers, to show a progress bar etc.

function: getContent(idArray, url, partLength, onPartSuccess, onFinalSuccess)

Start requesting content
param array idArray:
   an array with ids, that the server gets. If you need 100 units of somethings, the array has the id to any unit
param string url:
   that shall be contacted with the requests
param integer partLength:
   the size of requested units with every request. If you have an idArray with 100 units and partLength is 5, this will result in 20 requests then
param function onPartSuccess:
   a function that is called after each request
param function onFunalSuccess:
   a function that is called after alle requests are done
The example (Watch) uses php, which just answers with a debug string of the given ids:
1<?php 
2    $ids = $_POST['idArray'];
3    print_r($ids);
4?>
