Literal: KeyAction

Download KeyAction.js
since: 0.20062111
requires: prototype
author: Daniel Oltmanns
version: 0.20080823

Return the key code of an key event and do not let me search, which key it was.
Our example just put out some returned values (Watch):

function: getKeyCode()

Returns the keyCode of the key action.

function: isEnter()

Returns true, if the key event is the Enter-key.

function: isTab()

Returns true, if the key event is the Tab-key.

function: isControl()

Returns true, if the key event is a control-key (Tab, Shift, Enter, Alt etc.).
