Class: ScrollbarObserver

Download ScrollbarObserver.js
since: 0.20080502
requires: prototype
requires: ScrollingOffset
author: Daniel Oltmanns
version: 0.20080823
copyright: KNURT Systeme
license: GPL 3
This is to observer the scrollbars and to define functions on change.
The example just puts out the positions and you can start and stop observing (Watch):
1<html>
2    <head>
3        <title>ScrollbarObserver Demo</title>
4        <script type="text/javascript" src="pathto/prototype.js"></script>
5        <script type="text/javascript" src="pathto/jsscrollwork/ScrollingOffset.js"></script>
6        <script type="text/javascript" src="pathto/jsscrollwork/ScrollbarObserver.js"></script>
7        <script type="text/javascript">
8            window.onload = function() {
9                var so = new ScrollbarObserver(100, {
10                    onYChange : function(y) {
11                        document.getElementById('posy').innerHTML = y;
12                    },
13                    onXChange : function(x) {
14                        document.getElementById('posx').innerHTML = x;
15                    },
16                    onChange : function(x, y) {
17                        document.getElementById('posxy').innerHTML = '('+x+','+y+')';
18                    }
19                });
20                $('stop').onclick = function(){so.stop()};
21                $('start').onclick = function(){so.start()};
22            }
23        </script>
24    </head>
25    <body>
26        <div style="position: fixed">
27            Pos Y: <span id="posy"></span> Pixel<br />
28            Pos X: <span id="posx"></span> Pixel<br />
29            Pos (X,Y): <span id="posxy"></span> Pixel<br />
30            <button id="stop">Stop observing</button>
31            <button id="start">Start observing</button>
32        </div>
33        <div style="height: 10000px; width: 10000px;"></div>
34    </body>
35</html>

function: initialize(interval[, options])

Create a new instance of ScrollbarObserver and starts observing the scrollbars.
param int interval:
   the scrollbars are checked every "interval" ms
param function options.onXChange:
   define a function of what will happen if the horizontal scrollbar changes. This function gets the horizontal position of the scrollbar in pixel.
   new ScrollbarObserver(100, {onXChange: function(x){document.body.innerHTML = x};});
param function options.onYChange:
   define a function of what will happen if the vertical scrollbar changes. This function gets the vertical position of the scrollbar in pixel.
   new ScrollbarObserver(100, {onYChange: function(y){document.body.innerHTML = y};});
param function options.onChange:
   define a function of what will happen if the horizontal or vertical scrollbar changes. This function gets the horizontal and vertical position of the scrollbar in pixel.
   new ScrollbarObserver(100, {onChange: function(x, y){document.body.innerHTML = x + " " + y};});

function: setOnXChange(onXChange)

Set a (new) function of what will happen if the horizontal scrollbar changes.
param function onXChange:
   a function executed on horizontal scrollbar changes. This function gets the horizontal position of the scrollbar in pixel.
   so->setOnXChange(function(x){document.body.innerHTML = x};);

function: setOnYChange(onChange)

param function onYChange:
   a function executed on vertical scrollbar changes. This function gets the vertical position of the scrollbar in pixel.
   so->setOnYChange(function(y){document.body.innerHTML = y};);
Sets a (new) function of what will happen if the vertical scrollbar changes.

function: setOnChange(onChange)

Set a (new) function of what will happen if the horizontal scrollbar changes.
param function onChange:
   a function executed on horizontal or vertical scrollbar changes. This function gets the horizontal and vertical position of the scrollbar in pixel.
   so->setOnChange(function(x, y){document.body.innerHTML = x + " " + y};);

function: start()

Start observing the scrollbars.

function: stop()

Stop observing the scrollbars.
