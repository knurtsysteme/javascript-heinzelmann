Literal: FormCookieSaver

Download FormCookieSaver.js
since: 0.20080311
requires: prototype
requires: prototypeUtils
author: Daniel Oltmanns
version: 0.20080823
copyright: KNURT Systeme
license: GPL 3
Set form values of a page automaticly to values saved in a cookie. Save all form values (but password fields) into the same cookie for 1 year. If the page is visited a second time though, the formvalues are set out of the cookie automaticly.
Values can be set bei using FormCookieSaver.init() - mostly, on as follows:
1window.onload = function() {
2    FormCookieSaver.init();
3}
This sets all form elements of the page. If you just want specific form elements, set an array with all names of form elements as follows:
1window.onload = function() {
2    FormCookieSaver.init(['select2', 'name', 'notice']);
3}
Only form elements with name 'select2', 'name' and 'notice' will be set here.
ATTENTION: The name must be unique! Otherwise it will be ignored!
By default, one cookie per page is set. You can change this default behaviour by set a second param to init-method to true ...
1window.onload = function() {
2    FormCookieSaver.init(['select2', 'name', 'notice'], true);
3}
... sets 'select2', 'name' and 'notice' on all pages that reach this cookie or use explicitly ...
1window.onload = function() {
2    FormCookieSaver.init(null, true);
3}
... to set all form elements on all pages that reach this cookie.
If no cookie is set for this page, nothing happens. To set the cookie, just use FormCookieSaver.save() without params:
1window.onunload = function() {
2    FormCookieSaver.save();
3}
All saved form values must be init first.

function: init([formitemnames, globalsaving])

Initialize the FormCookieSaver:
param array formitemnames:
   the form items, that shall be saved into the cookie (Default: All form elements in the document)
param boolean globalsaving:
   set this to true, if the cookie shall be saved the biggest location possible (not only the current page) and set to false, if form values shall only be saved for the current page (Default: false).
Example:
1window.onload = function() {
2    FormCookieSaver.init(['select2', 'name', 'notice'], true);
3}

function: save()

Saves all actual values initialized into the cookie - in the example, the event window.onunload is used.
1window.onunload = function() {
2    FormCookieSaver.save();
3}
