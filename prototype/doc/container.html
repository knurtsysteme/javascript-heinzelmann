If you have a very big DOM, prototypes $$('.classname') might take some time. This saves elements of specific class names in a Container, so internly, prototypes $$('.classname') is just called once per classname.
In 99.9 % of cases - you won't need this! But if you have such a big DOM, this might help.

function: getElementsByClassName(className, renew)

Return all elements with the given class name and save it into the container.
param string className:
   the name of the class attribute of the elements, you want work with an that are automaticly saved in the container
param boolean renew:
   if you want to renew all container elements (e.g. because of your DOM has been changed), use this function with renew = true. This is optional
If there are already elements with this classname in the container, return that.
This function is NOT deprecated as prototypes Element.getElementsByClassName is, because internly prototypes $$ is used.
