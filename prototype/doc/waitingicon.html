Literal: WaitingIcon

Download WaitingIcon.js
since: 0.20081210
requires: prototype
requires: Browser
requires: ScrollingOffset
author: Daniel Oltmanns
version: 0.20080823
copyright: KNURT Systeme
license: GPL 3
Show a waiting item global on a page (it is not possible to show this in special block elements or more then one waiting icon at the same time). The waiting Icon is whatever-you-want-html in always shown in the center of your page.
The example shows a waiting icon (a simple headline) for 3 seconds (Watch):
1<html>
2    <head>
3        <title>WaitingIcon Demo</title>
4        <script type="text/javascript" src="pathto/prototype.js"></script>
5        <script type="text/javascript" src="pathto/jsscrollwork/Browser.js"></script>
6        <script type="text/javascript" src="pathto/jsscrollwork/ScrollingOffset.js"></script>
7        <script type="text/javascript" src="pathto/jsscrollwork/WaitingIcon.js"></script>
8        <script type="text/javascript">
9            window.onload = function() {
10                WaitingIcon.init('<h1>PLEASE WAIT</h1>');
11                WaitingIcon.show();
12                window.setTimeout(WaitingIcon.hide, 3000); // show it 3 seconds
13            }
14        </script>
15    </head>
16    <body></body>
17</html>
This example uses a simple window.setTimeout to hide the icon after 3 seconds. Even if you hide it without that timeout (WaitingIcon.show(); WaitingIcon.hide();), it is always shown at minimum ms (Default 2 seconds), because it would be annoying, if the user just see the icon for a couple of miliseconds and you do not always know, how long the waiting icon will be shown (e.g. in case of an ajax request).

function: init(html)

initilize the waiting icon
param string html:
   The html to show (mostly a gif animation)

attribute: id

id of the waiting icon (Default "knurt_waitingicon")

attribute: waitingElement

the element with the waiting icon (Before init null)

attribute: showMinimumTimeMs

Time in miliseconds, how long the icon is shown at least

function: show()

Show the waiting icon

function: hide()

Hide the waiting icon
