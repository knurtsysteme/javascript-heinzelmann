Welcome to JavaScript Scrollwork

This documentation is for an earlier release. The functionality is nearly the same. Anyway: It may vary.
Since the 2.00-SNAPSHOT, all functions can be found in one single file called scrollwork.js.
JavaScript Scrollwork is a collection of goodies, making life easier and other things, that pimp your webapplication or webpage.
Some JavaScripts based on any of the following Librarys (see "requires" on the doc-pages of the Literal or Class):
  • prototype
  • a changed version of prototypeUtils (Original)
In the download package of JavaScript Scrollwork, you find them all in the directory dependencies.
