Class: AnchorTagger

Download AnchorTagger.js
since: 0.20080823
requires: prototype
author: Daniel Oltmanns
version: 0.20080823
copyright: KNURT Systeme
license: GPL 3
This calls a function on all local or foreign anchors (HTMLAnchorElement) on the entire page or on a part of the page.
Example 1: Give all foreign anchors on your page a green color:
1var at = new AnchorTagger();
2at.tagForeign(function(el){el.setStyle({color: 'green'});});
Example 2: Hide all your local anchors in your menu (a div-element with the id 'menu'):
1var at = new AnchorTagger({area: $('menu')});
2at.tagLocal(Element.hide);

constructor: initialize([options])

Construct an AnchorTagger.
param element area:
   the area in the DOM, that links shall be tagged. (Default: the entire document)
Example 1: new AnchorTagger()
Example 2: new AnchorTagger({area: $('myarea')});

attribute: foreignAnchors

An Array of all your <a> elements, that links on another site but yours.

function: isLocalAnchor(el)

Return true, if the given <a>-element el links on your site. Otherwise false.
param HtmlAnchorElement el:
   that shall be checked

function: tagLocal(func)

Call the given function func on all the local anchors.
param function func:
   that is executed on all local anchors

function: tagForeign(func)

Call the given function func on all the foreign anchors.
param function func:
   that is executed on all foreign anchors
