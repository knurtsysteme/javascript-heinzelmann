function: is.nothingOrWhitespace(string)

If the entire input is the empty string or just whitespace, return true. Otherwise false.
param string string:
   The string, that will be checked

function: is.regularEmail(string)

If the entire input is a regular e-mail adress, return true. Otherwise false.
param string string:
   The string, that will be checked
There are people, that discusses regular expressions for checking e-mails the whole day long. To make it easy, here is the (not perfect but for most purposes enough) regular expression used here: ^.+@.+..+$

function: is.date_YYYYMMDD(string)

If the entire input is a date in form of YYYYMMDD (gregorian calendar), return true. Otherwise false. Just dates between 1st of January 1000 and 31st of December 9999 are checked. Heap years are noticed as well.
param string string:
   The string, that will be checked
Some examples:
  • 20080101: true
  • 20080131: true
  • 20080431: false (april just have 30 days)
  • 20080229: true (2008 is a heap year)
  • 20070229: false (2007 is not a heap year)
  • 20081301: false (13th month does not exist)
  • 9990229: false (This works from jear 1000 on)
  • 100000229: false (This works just to jear 9999 on)

function: contains.whitespace(string)

If the input contains whitespace, return true. Otherwise false.
param string string:
   The string, that will be checked
